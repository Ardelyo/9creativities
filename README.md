# Nine Creativities

Nine Creativities adalah platform web inovatif yang menampilkan proyek-proyek kreatif dan teknologi yang dikembangkan oleh siswa-siswi kelas 9C. Website ini menggabungkan desain modern dengan fungsionalitas yang kuat untuk memberikan pengalaman pengguna yang menarik dan informatif.

## Fitur Utama

1. **Halaman Beranda (Index):**
   - Menampilkan empat kartu navigasi utama: Teknologi Lingkungan, Teknologi Pangan, Teknologi Informasi, dan Teknologi Komunikasi.
   - Animasi Framer Motion untuk transisi yang halus.
   - Latar belakang gradien dinamis dengan elemen dekoratif.

2. **Halaman Tim:**
   - Setiap tim memiliki halaman dedikasi dengan:
     - Pengenalan tim dan fokus proyek.
     - Profil anggota tim (foto, nama, peran, minat).
     - Deskripsi detail proyek utama.
     - Galeri foto proyek dan dokumentasi.
   - Desain responsif untuk berbagai ukuran layar.

3. **Fitur Robot C (Chatbot):**
   - Antarmuka chat interaktif dengan AI assistant.
   - Menggunakan API Gemini untuk respons yang relevan dan informatif.
   - Fitur pertanyaan lanjutan untuk memperdalam pemahaman.

4. **Navigasi dan Desain:**
   - Komponen DynamicIsland untuk navigasi utama.
   - Animasi transisi halaman dengan Framer Motion.
   - Tema warna konsisten untuk setiap tim.

5. **Teknologi yang Digunakan:**
   - React untuk pengembangan frontend.
   - Tailwind CSS untuk styling.
   - Framer Motion untuk animasi.
   - React Router untuk navigasi.
   - Komponen UI dari Shadcn/ui.

6. **Responsivitas dan Aksesibilitas:**
   - Desain fully responsive.
   - Penggunaan atribut ARIA untuk aksesibilitas.

7. **Performa:**
   - Optimisasi gambar dan aset.
   - Lazy loading untuk komponen dan gambar.

## Konsep Robot C

Robot C adalah fitur chatbot AI yang terintegrasi dalam website Nine Creativities. Berikut adalah penjelasan detail tentang konsep dan teknologi yang digunakan:

1. **Tujuan:**
   Robot C dirancang untuk memberikan informasi interaktif kepada pengunjung website tentang proyek-proyek Nine Creativities dan topik teknologi terkait.

2. **Antarmuka Pengguna:**
   - Tampilan chat yang intuitif dan responsif.
   - Input teks untuk pertanyaan pengguna.
   - Tombol kirim untuk mengirim pesan.
   - Area tampilan chat untuk menampilkan percakapan.

3. **Teknologi Utama:**
   - **React:** Digunakan untuk membangun komponen UI yang interaktif dan dinamis.
   - **Tailwind CSS:** Memberikan styling yang konsisten dan responsif.
   - **Framer Motion:** Menyediakan animasi smooth untuk transisi dan interaksi UI.
   - **API Gemini:** Menggunakan model bahasa AI canggih dari Google untuk menghasilkan respons.

4. **Fitur-fitur Teknis:**
   - **Manajemen State:** Menggunakan React hooks (useState, useEffect) untuk mengelola state percakapan.
   - **Asynchronous API Calls:** Menggunakan fetch API untuk berkomunikasi dengan Gemini API.
   - **Markdown Rendering:** Menggunakan react-markdown untuk menampilkan respons terformat.
   - **Scroll Otomatis:** Implementasi useRef untuk scroll otomatis ke pesan terbaru.

5. **Keamanan:**
   - Input API key dari pengguna untuk autentikasi dengan Gemini API.
   - Penyimpanan API key secara lokal (tidak dikirim ke server).

6. **Konteks Khusus:**
   Robot C diprogram dengan pengetahuan khusus tentang Nine Creativities, termasuk informasi tentang tim-tim proyek dan inovasi mereka.

7. **Pertanyaan Lanjutan:**
   - Fitur yang menganalisis respons AI untuk menghasilkan pertanyaan lanjutan yang relevan.
   - Membantu pengguna memperdalam pemahaman mereka tentang topik yang dibahas.

8. **Optimisasi Performa:**
   - Lazy loading untuk komponen chat.
   - Throttling pada input pengguna untuk mengurangi beban API.

9. **Aksesibilitas:**
   - Penggunaan atribut ARIA untuk memastikan chatbot dapat diakses oleh pengguna dengan kebutuhan khusus.
   - Kontras warna yang baik untuk keterbacaan.

10. **Responsivitas:**
    - Desain adaptif yang menyesuaikan dengan berbagai ukuran layar (mobile, tablet, desktop).
    - Penggunaan Flexbox dan Grid untuk layout yang fleksibel.

Robot C merupakan contoh integrasi teknologi AI modern dalam konteks pendidikan, memungkinkan interaksi yang lebih mendalam dengan konten website dan memperkaya pengalaman belajar pengunjung Nine Creativities.

## Teknologi Informasi

Teknologi Informasi (TI) adalah bidang yang mencakup penggunaan komputer dan sistem digital untuk menyimpan, mengambil, mengirim, dan memanipulasi data atau informasi. Dalam konteks Nine Creativities, tim Teknologi Informasi bertanggung jawab untuk:

1. **Pengembangan Website:**
   - Menggunakan React untuk membangun antarmuka pengguna yang dinamis dan responsif.
   - Implementasi Tailwind CSS untuk styling yang efisien dan konsisten.
   - Penggunaan Framer Motion untuk animasi halus dan transisi menarik.

2. **Manajemen Data:**
   - Implementasi sistem manajemen state menggunakan React hooks.
   - Penggunaan React Query untuk fetching dan caching data efisien.

3. **Optimisasi Performa:**
   - Implementasi lazy loading untuk komponen dan gambar.
   - Optimisasi kode untuk memastikan kecepatan loading yang baik.

4. **Keamanan:**
   - Implementasi praktik keamanan terbaik dalam pengembangan web.
   - Penanganan input pengguna yang aman untuk mencegah serangan.

5. **Integrasi API:**
   - Mengintegrasikan API Gemini untuk fitur chatbot Robot C.
   - Mengelola panggilan API asinkron untuk pengalaman pengguna yang lancar.

Proyek utama tim ini adalah pengembangan website Nine Creativities itu sendiri, yang berfungsi sebagai platform untuk memamerkan inovasi dari semua tim.

## Teknologi Pangan

Teknologi Pangan di Nine Creativities berfokus pada pengembangan produk pangan inovatif dengan manfaat kesehatan. Aspek-aspek utama meliputi:

1. **Fermentasi Makanan:**
   - Pengembangan teknik fermentasi untuk meningkatkan nilai gizi dan umur simpan makanan.
   - Penelitian tentang probiotik dan manfaat kesehatan terkait.

2. **Pengolahan Pangan:**
   - Implementasi teknologi pengolahan modern untuk mempertahankan nutrisi.
   - Pengembangan metode pengawetan alami.

3. **Inovasi Produk:**
   - Menciptakan makanan fungsional dengan manfaat kesehatan tambahan.
   - Eksperimen dengan bahan-bahan lokal untuk menciptakan produk unik.

4. **Keamanan Pangan:**
   - Implementasi sistem manajemen keamanan pangan.
   - Pengembangan metode pengujian untuk memastikan kualitas produk.

5. **Nutrisi Terapan:**
   - Analisis komposisi nutrisi produk pangan.
   - Optimalisasi formula untuk meningkatkan profil gizi.

Proyek utama tim ini adalah Produksi Yoghurt Probiotik, yang menggabungkan pengetahuan fermentasi, nutrisi, dan teknologi pengolahan pangan untuk menciptakan produk yang sehat dan lezat.

## Teknologi Lingkungan

Tim Teknologi Lingkungan di Nine Creativities fokus pada pengembangan solusi untuk masalah lingkungan, terutama terkait pengelolaan air. Aspek-aspek utama meliputi:

1. **Pemurnian Air:**
   - Pengembangan sistem filtrasi air menggunakan bahan-bahan alami dan berkelanjutan.
   - Implementasi teknologi membran untuk pemurnian air tingkat lanjut.

2. **Konservasi Air:**
   - Perancangan sistem pengumpulan dan pemanfaatan air hujan.
   - Pengembangan metode irigasi hemat air untuk pertanian urban.

3. **Pengolahan Air Limbah:**
   - Implementasi sistem pengolahan air limbah skala kecil.
   - Penelitian tentang fitoremediasi untuk membersihkan air tercemar.

4. **Monitoring Kualitas Air:**
   - Pengembangan sensor dan sistem IoT untuk pemantauan kualitas air real-time.
   - Analisis data untuk prediksi dan pencegahan pencemaran air.

5. **Edukasi Lingkungan:**
   - Merancang program edukasi tentang konservasi air dan lingkungan.
   - Mengembangkan alat peraga interaktif untuk menjelaskan konsep lingkungan.

Proyek utama tim ini adalah Filter Air Ramah Lingkungan, yang menggunakan kombinasi bahan alami untuk menciptakan sistem pemurnian air yang efektif dan berkelanjutan.

Melalui integrasi keempat bidang teknologi ini - Informasi, Komunikasi, Pangan, dan Lingkungan - Nine Creativities menciptakan ekosistem inovasi yang komprehensif, mempersiapkan siswa untuk menghadapi tantangan teknologi dan lingkungan di masa depan.
